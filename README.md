만든 boilerplate를 이용하여 api crud를 연습해보았습니다.  
boilerplate-with-cra 를 이용하였습니다.

- 공부하여 보완하고싶은 사항 1 : 이벤트
- 공부하여 보완하고싶은 사항 2 : 번호를 직접 입력하지 않고, 테이블에서 클릭하여 수정/삭제
